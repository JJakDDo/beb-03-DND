# Server

## 패키지 설치

```bash
npm install
```

### 환경변수 설정

#### HTTP Server<br>
PORT=서버 접속 포트번호

#### MongoURI<br>
USER_NAME=Atlas cluster 계정<br>
USER_PASSWORD=Atlas cluster 계정 비밀번호<br>
CLOUD_NAME=Atlas cluster 이름<br>
DATABASE_NAME=Atlas cluster에서 생성한 Database 이름


## HTTP Server 연결

```bash
npm start
```
